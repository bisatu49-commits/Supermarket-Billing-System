# Supermarket-Billing-System

## Project Description
The SmartBuy Supermarket Billing System is a Python program that automates the billing process for SmartBuy Supermarket in Freetown, Sierra Leone. It replaces manual calculations with an automated system that is fast, accurate, and professional.

## How the Program Works
1. Cashier launches the program
2. Enters product name, quantity, and price for each item
3. Types 'done' when finished entering products
4. System calculates the total bill automatically
5. A 10% discount is applied if total exceeds Le 500
6. A formatted receipt is displayed
7. Cashier can process multiple customers

## How to Run the Program
1. Install Python 3 on your computer
2. Download the file supermarket_billing.py
3. Open terminal and type: python supermarket_billing.py
4. Follow the on-screen instructions

## Features
- Accepts multiple products per customer
- Stores products in a list (array)
- Calculates totals automatically
- Applies 10% discount for purchases over Le 500
- Displays a professional receipt
- Processes multiple customers in one session
